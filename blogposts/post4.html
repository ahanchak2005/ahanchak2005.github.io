A cool NT problem

PROBLEM: Let $ a $ and $ b $ be positive integers. Prove that the number of solutions $ (x,y,z)$ in non-negative integers to the equation $mx+ ny+z = mn$ is

$ \frac{1}{2}\left[ (m+1)(n+1)+\text{gcd}(m,n)+1\right]$

(NOT NEARLY A FULL BUT VERY UGLY) Solution with NT :

We use the following lemma (also called the Gauss' Reciprocity Lemma) that if $p,q$ are coprime integers then

$$\sum\limits_{k=0}^{q-1}\lfloor\frac{kp}{q}\rfloor = \sum\limits_{k=0}^{p-1}\lfloor \frac{kq}{p}\rfloor = \frac{(p-1)(q-1)}{2}$$

We can prove this lemma by simply noting that $kp$ are mutually incongruent in $\pmod q$ where $k\in\mathbb{N}, 0\le k \le q-1$. Now so the set of the numbers $\{\frac{kp}{q}\}$ are in some order the set of the numbers $\frac{k}{q}$. That proves the lemma.

Now if we fix $x=k$ then note that $0\le y \le \lfloor\frac{m(n-k)}{n}\rfloor$. So we get the number of such possible $y$ s for a fixed value of $x$( We can always take $z$ accordingly.

Now note that we can calculate the sum by using the previous lemma. If $\text{gcd}(m,n)=1$ then the problem is almost done. For other cases, the calculations will be a little lengthier(or a lot).

A Combinatorics Solution :

Note that finding number of the triplets of non-negative integers so that they satisfy the equation is equivalent to finding the number of pairs of non-negative integers $(x,y)$ such that they satisfy the inequality $mx+ny\le mn$.
Now note that it is equivalent to finding the number of lattice points inside or on the region enclosed in the FIRST QUADRANT by the co-ordinate axes and the straight line $mx+ny=mn \text{ or, } \frac{x}{n}+\frac{y}{m} = 1$. Now say that number of such points which lie on the line $mx+ny=mn$ be $l$ and which are not on the line but lie in the region specified be $a$. Now note that by symmetry on the other half-rectangle ( not on the line ) the number of such points is also $a $. So , we know that $2a+l = (m+1)(n+1)$ (As choice of $(x,y)$ runs from $0$ to $m$ and from $0$ to $n$ respectively.

Now we want to find $l$. We take that $\text{gcd}(m,n) = g$ and say $m=gm';n=gn'$ and also note that $m',n'$ are coprime. Now our new equation is $m'x+n'y= gm'n'$ Now note that $m'|n'y\implies m'|y$ as $m',n'$ are coprime. Say , $y=m'k$ for some integer $k$. Also note that as $0\le y\le m'g$ so number of choices for $k $ is $g+1$. Also note that fixing $k $ fixes $y=m'k$ and by diving both side by $m'$ we get that fixing $k$ also fixes $x$, Hence fixing $k$ fixes the pair. As number of choices of $k$ is $g+1$. So number of such pair $(x,y)$ so that $mx+ny = mn$ is $g+1$. Hence, $l=g+1$

Hence $2a+2l = (m+1)(n+1)+g+1\implies \boxed{ a+l = \frac{(m+1)(n+1)+g+1}{2}}$
